Personality Generator

Last Updated: 12/27/2017

Random personality traits generator to help me come up with personalities for characters. I was not able to get an API call to a dictionary to fetch word definitions, so instead I made a .js file with a 2D array of words and definitions.
NES-Style Sprite Generator

Last Updated: 12/27/2017

I modified a pre-existing sprite generator as an exercise in character design for me. I click the generate button, and I draw what I think the sprite empbodies. Inspired by old NES graphics that didn't have a lot of detail to them, while full-page artwork in magazines filled in the design details (especially the arcade game Ladybug, pictured above). I plan on adding some examples of artwork that I've made.
